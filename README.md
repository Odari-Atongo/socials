# Socials
This is a CRUD social media type application created using the MERN stack

# Technologies used
 - React
 - Express
 - Nodejs
 - MongoDB

# Installation
To get started, you need to clone this repo into your local machine, and run the following commands

```
cd socials
npm install
npm start
```
